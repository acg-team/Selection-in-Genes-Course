Software

The software below will be used for the exercises.

  • PhyML: PhyML is a software package that uses modern statistical approaches to analyse alignments of nucleotide or amino acid sequences in a phylogenetic framework (download).
  • PAML: PAML is a package of programs for phylogenetic analyses of DNA or protein sequences using maximum likelihood (download).
  • FigTree: FigTree is a graphical viewer of phylogenetic trees (download).
  • AliView: AliView is an alignment viewer and editor (download).
